# Arduino basic
--- 

### Basic

`//` - однострочный комментарий\
`/* ... */` - многострочнй комментарий
> комментарии не занимают память после компиляции\

`#include <Lib>` - подключение файлов(библиотек)\
`void setup () {}` - функция инициализации(выполнется единажды при запуске)\
`void loop () {}`  - бесконечный цикл\
`type var_name;` / `type var_name = value;` - обьявление переменной с обязательным указанием типа данных

### Типы данных

| Название | Вес | Диапазон | Описание |
| --- | --- | --- | --- |
| **boolean** | 1B | 0 или 1| Догическая переменная, может принимать значение *true(1)* или *false(0)* |
| **char** | 1B | -128 ... 128 | Хранит номер символа из таблици символов ASCII |
| __byte__| 1B | 0 ... 255 |  |
| __int__ | 2B  | -32768 ... 32767 ||
| __unsigned int__ | 2B |  0 ... 65535 ||
| __word__ | 2B |  0 ... 65535 ||
| __long__| 4B | -2\*10\*\*9 ... 2\*10\*\*9 ||
| __unsigned long__ | 4B | 0 ... 4*10**9 ||
| __float__ | 4B |  | Хранит числа с плавающей точкой |
> __float__ 
> - присваивать только значение с точкой, даже если оно целое(2.0)
> - Делить тоже только на число с точкой, даже если они целые( var / 2.0 )
> - `float result = (float)int_var / float` - деление целого на число с точкой

### Serial

`Serial` - обект библиотеки Serial для работы с последовательным портом\
`Serial.begin(9600);` - открытие портa\
`Serial.print("Текст");` - вывод в порт\
`Serial.println(var);` - вывод с переводом строки\
`Serial.print(pi, 3);` - вывод *float* с указанием количества знаков после запятой\
`Serial.available();` - проверить буфер на наличие входящих данных(max 64B). Возвращает количество байт\
`Serial.read();` - чтение данных из буфера(ASCII)\
`Serial.read() - 0` - для форматирования ASCII в число\
`Serial.parseint()` - получить число из нескольких цифр

### Условный оператор, операторы сравнения, логические операторы

`if () {}` - условный оператор, проверяет условие в `()`. Если условие верно, то выполняется `{}`\
**Операторы сравнение**: `a == b`(равно), `a != b`(не равно), `>`(строго больше), `<`(строго меньше), `>=`(больше или равно), `<=`(меньше или равно)\
**Логические операторы**: `&&`(и), `||`(или), `!`(не)\
```
if () {

} else if {

} else {

}
```

```
switch (val) {
    case val_1: ... ;
        break;
    case val_2 : ... ;
        break;
    ...
    default: ... ;
}
```
